This is a mission to stacking cubes.

You are given a list of cube details (tuple of 3 integers: X coordinate, Y coordinate, edge length).
You have to return the largest in stacking height of the cubes.

  • Each coordinate is the minimum value.
  • All edges parallel to the coordinate axes.
  • The order of stacking the cubes is arbitrary, but the X and Y coordinates of the cubes can't be changed.
  • if at least 1x1 faces touch, they will be stacked.

Note :

  • You don't always need to use the all cubes.

Examples :

stacking_cubes([(0, 0, 2), (1, 1, 2), (3, 2, 2)]) == 4
stacking_cubes([(0, 0, 2), (1, 1, 2), (0, 2, 2), (2, 2, 1)]) == 6
stacking_cubes([(0, 0, 2), (0, 2, 2), (0, 2, 2), (2, 0, 2), (2, 0, 2), (2, 0, 2), (2, 0, 2)]) == 8

Input : A list of tuples of 3 integers.

Output : An integer.

Precondition :

  • 2 ≤ len(cubes) ≤ 50
  • -100 ≤ X, Y coordinate ≤ 100
