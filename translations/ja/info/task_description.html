立方体を積み重ねるミッションです。

あなたは立方体の詳細 (3つの整数のタプル : X座標, Y座標, 辺の長さ) のリストを与えられます。 与えられた立方体をできるだけ高く積み重ねた結果の高さを整数で返して下さい。

  • 各座標は最小値です。全ての辺は座標軸に平行です。
  • 立方体を積み重ねる順番は任意ですが立方体の X, Y座標は変更出来ません。
  • 少なくとも 1x1 の面が接していれば積み重ねられます。

注意 :

  • 常に全ての立方体を使う必要はありません。

Examples :

stacking_cubes([(0, 0, 2), (1, 1, 2), (3, 2, 2)]) == 4
stacking_cubes([(0, 0, 2), (1, 1, 2), (1, 2, 1), (2, 2, 2)]) == 6
stacking_cubes([(0, 0, 2), (0, 2, 2), (0, 2, 2), (2, 0, 2), (2, 0, 2), (2, 0, 2), (2, 0, 2)]) == 8

入力 : 立方体の詳細(3つの整数のタプル : X座標, Y座標, 辺の長さ) のリスト

出力 : 積み重ねた立方体の高さ(整数)

前提条件:

  • 2 ≤ 立方体の数 ≤ 50
  • -100 ≤ X(Y) 座標 ≤ 100
